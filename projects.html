RPG-Like Walking Sim

From 2021 to 2022, I worked on a multimedia project to pursue a combination interests and snowball a few more. As a part of that project, I spent a month building a top-down RPG style walk-around segment through Godot and its proprietary language, GDScript.

Weighted Conversation System

In 2020, I had an idea to make an RPG that utilized a weighted conversation system in an attempt to make the dialog come more naturalistically--that is, in non-linear, yet logical order. The scripting for this was done in RMMV, which taught me the core ideas of Node.js, and the potential hazards when multiple scripters target the same functions in the core engine. This code is available on request.

Tactical Artillery Game

A project idea I had for a game like Worms, Tanks or Arcanists that I worked on from 2017 to 2018. Though unfinished, I built a state-pattern-based movement system in C# within Unity that provides detailed information on the player's current state.

Discord Bot

Discord bot I made to support an interactive webcomic project built off and on between 2017 and 2018. Provided my first, practical experience with Python and working with asynchronous data. Built an interface that made use of reactions as buttons when that was still rare.
