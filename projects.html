RPG-Like Walking Sim

From 2021 to 2022, I worked on a multimedia project to pursue a combination interests including illustration, storyboarding and web design. As a part of that project, I spent a month building a top-down RPG style walking sim through Godot and its proprietary language, GDScript. Working through this was helpful in teaching me not to reinvent the wheel and to make use of existing components instead of trying to build everything from scratch as in my previous Unity project.

Conversation Card Game Prototype

In 2021, I was inspired by Signs of the Sojourner to try to create a conversation-based card game to act as a more ludonarrative way to manage important RPG dialog. The result was rather clunky, but I learned quite a bit about prototyping and receiving feedback. I've even got some ideas, now, on how to improve the idea when I pick it up again.

Weighted Conversation System

In 2020, I had an idea to make an RPG that utilized a weighted conversation system to make dialog more naturalistical--that is, in non-linear, yet logical order. The scripting for this was done in RMMV, which taught me the core ideas of Node.js, and the potential hazards when multiple scripters target the same functions in the core engine. This code is available on request.

Tactical Artillery Game

A project idea I had for a game like Worms, Tanks or Arcanists that I worked on from 2017 to 2018. Though unfinished, I built a state-pattern-based movement system in C# within Unity that provides detailed information on the player's current state.

Discord Bot

Discord bot I made to support an interactive webcomic project built off and on between 2017 and 2018. Provided my first, practical experience with Python and working with asynchronous data. Built an interface that made use of reactions as buttons when that was still rare.
